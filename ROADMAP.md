# **SUBIT‑64 Roadmap**  
### Evolution plan for the SUBIT semantic system and tooling

This roadmap outlines the planned development of the SUBIT‑64 ecosystem across core semantics, tooling, interfaces, and applied modules.  
The structure follows a layered, fractal expansion: **Core → Tools → Interfaces → Applications → Ecosystem**.

---

## **1.0 — Completed (Public Release)**  
### Core
- Full SUBIT‑64 semantic atlas (64 archetypes)
- Canonical English dictionary (mask, shadow, optimum, pathology)
- YAML and JSON dictionary formats
- Mapping examples dataset

### Tools
- Minimal interpretable classifiers:
  - `subit-mapper.py`
  - `subit-mapper.js`
  - `subit-mapper-web.js`

### Documentation
- `cli-demo.md`
- `cli-demo-advanced.md`
- `CHANGELOG.md`
- Initial repository structure

# **1.5 — Guiding Principles**
- **Minimalism** — no unnecessary complexity  
- **Transparency** — interpretable logic, no black boxes

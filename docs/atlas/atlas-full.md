# SUBIT‑64 ATLAS  
## Complete Archetype Dictionary (v1.0)

SUBIT‑64 ATLAS is a universal semantic system consisting of **64 archetypes**, organized into **8 groups** that form the SUBIT System Stack — a structured model of how meaning unfolds from matter to abstraction.

Each archetype includes:

- **Mask** — the core form  
- **Behavior** — how the archetype acts  
- **Shadow** — its negative or distorted mode  
- **Optimum** — healthy expression  
- **Pathology** — dysfunctional expression  
- **Applications** — where the archetype is used  

This document contains the **entire Atlas in one file**.

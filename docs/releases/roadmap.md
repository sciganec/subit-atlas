# SUBIT‑64 ATLAS — Roadmap  
## Development Plan, Milestones, and Future Extensions

This roadmap outlines the **strategic direction** for SUBIT‑64 following the release of **v1.0**.  
It defines upcoming milestones, research tracks, product expansions, and long‑term architectural goals.

The roadmap is divided into:

- **Short‑term (v1.1–v1.2)** — consolidation, tooling, integrations  
- **Mid‑term (v2.x)** — extensions, domain packs, visualization  
- **Long‑term (v3.x+)** — reasoning engines, multi‑modal systems, SUBIT‑128  

This document is intentionally modular and may evolve as the ecosystem grows.

---

# 1. Short‑Term Roadmap (v1.1–v1.2)  
## Focus: Stability, Tooling, Developer Experience

### **1.1 Documentation Enhancements**
- Add diagrams for System Stack and Axes  
- Provide cross‑group comparison tables  
- Add “SUBIT in Practice” case studies  
- Expand examples for multi‑label and trajectory mapping  

### **1.2 Developer Tools**
- CLI classifier demo  
- Python and JavaScript SDKs  
- Local inference engine (rule‑based)  
- JSON schema validation for SUBIT data  

### **1.3 Machine‑Readable Assets**
- Add YAML format for archetypes  
- Provide embeddings aligned with SUBIT axes  
- Release topological adjacency matrix  

### **1.4 Classifier Improvements**
- Better ambiguity handling  
- Domain‑specific presets (emotion, UX, narrative)  
- Configurable stack constraints  

---

# 2. Mid‑Term Roadmap (v2.x)  
## Focus: Expansion, Visualization, Domain‑Specific Atlases

### **2.1 SUBIT‑Domain Packs**
Specialized sub‑atlases built on the 64‑state core:

- **SUBIT‑Emotion** (expanded Group 6)  
- **SUBIT‑Narrative** (story arcs, character states)  
- **SUBIT‑UX** (interaction patterns, user states)  
- **SUBIT‑Organizational** (roles, structures, dynamics)  
- **SUBIT‑Perception** (color, texture, sensory fields)  

Each pack includes:

- domain‑specific mapping rules  
- extended examples  
- optional micro‑archetypes  

### **2.2 Visualization Tools**
- Interactive 64‑node hypercube viewer  
- Stack‑layer visualizer  
- Axis‑based semantic map  
- Trajectory animation tool  

### **2.3 Reasoning Extensions**
- SUBIT‑based rule engine  
- Path prediction and semantic forecasting  
- Archetype clustering and similarity search  

### **2.4 Data Integrations**
- Knowledge graph export  
- Ontology alignment tools  
- Vector‑space alignment with ML embeddings  

---

# 3. Long‑Term Roadmap (v3.x+)  
## Focus: Advanced Reasoning, Multi‑Modal Systems, SUBIT‑128

### **3.1 SUBIT‑Reasoner**
A full reasoning engine built on:

- axis logic  
- topological transitions  
- stack‑aware inference  
- multi‑step semantic trajectories  

Capabilities:

- explainable reasoning  
- narrative generation  
- system modeling  
- conflict resolution  
- emotional simulation  

### **3.2 Multi‑Modal SUBIT**
Mapping across:

- text  
- images  
- audio  
- behavior logs  
- sensor data  

Unified semantic layer for multi‑modal AI.

### **3.3 SUBIT‑128 (Extended Space)**
A future expansion doubling the semantic resolution:

- 7 axes → 128 archetypes  
- higher granularity  
- domain‑specific micro‑states  
- backward compatibility with SUBIT‑64  

### **3.4 SUBIT‑Ecosystem**
- Plugins for major AI frameworks  
- Integration with simulation engines  
- SUBIT‑native narrative and world‑building tools  
- Organizational diagnostics and coaching systems

# 4. Research Tracks

### **4.1 Semantic Minimalism**
Investigate whether 64 is the minimal universal set.

### **4.2 Cross‑Cultural Archetype Mapping**
Align SUBIT with global symbolic systems.

### **4.3 Cognitive Modeling**
Use SUBIT as a model of human perception and reasoning.

### **4.4 System Dynamics**
Model complex systems as trajectories in SUBIT space.

# 5. Release Cadence

- **v1.x** — stabilization and tooling  
- **v2.x** — expansion and visualization  
- **v3.x** — reasoning and multi‑modal systems  
- **v4.x+** — extended semantic architectures  

Each major version maintains backward compatibility with the core 64 archetypes.

---

# 6. Summary

The SUBIT‑64 roadmap outlines a multi‑year evolution:

- **v1.x** — foundation and tooling  
- **v2.x** — domain packs and visualization  
- **v3.x** — reasoning engines and multi‑modal integration  
- **v4.x** — extended semantic architectures (SUBIT‑128)  

The long‑term vision is a **universal semantic layer** for AI, modeling, narrative, and human‑system interaction.

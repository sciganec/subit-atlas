# 2. SUBIT‑64 System Stack (v1.0)

The System Stack is the **vertical architecture** of SUBIT‑64:

1. **Matter** — physical substrate  
2. **Organic** — growth and biological logic  
3. **Dynamics** — movement and processes  
4. **Perception** — sensory qualities  
5. **Spectrum** — perceptual fields  
6. **Psychic** — internal states  
7. **Social** — collective structures  
8. **Abstract** — meta‑models and conceptual structures  

Each level is structurally dependent on the one below it.

# 3. Archetype Set (v1.0)

The 64 archetypes are stable and final for this release.

- **S32–S39** Spectrum

# 4. Semantic Axes (v1.0)

SUBIT‑64 is built on **six binary axes**:

1. Materiality ↔ Abstraction  
2. Unity ↔ Multiplicity  
3. Stability ↔ Change  
4. Internal ↔ External  
5. Order ↔ Chaos  
6. Passive ↔ Active  

These axes generate the 64‑state semantic space.

---

# 5. Topology (v1.0)

The topology is defined as:

- a **6‑dimensional hypercube**  
- 64 nodes  
- each node has 6 neighbors  
- adjacency defined by axis flips  
- groups form contiguous regions  
- transitions follow semantic gradients  

This topology is the backbone of SUBIT reasoning.

# **SUBIT‑64 Layer Dynamics**

This document describes the dynamic behavior of the eight layers of the SUBIT‑64 stack.  
While the stack defines *semantic altitude*, layer dynamics describe how each layer:

- transforms axis patterns  
- modulates transitions  
- shapes trajectories  
- amplifies or suppresses certain polarities  
- produces characteristic dynamic modes  

This document is **informative**, not normative.

---

# **1. Dynamic Role of Layers**

Each layer acts as a **semantic transformer**.  
Layers do not change axis definitions, but they change:

- how axes *express themselves*  
- how transitions *propagate*  
- how processes *unfold*  
- how states *stabilize or destabilize*  

Layers are dynamic filters.
